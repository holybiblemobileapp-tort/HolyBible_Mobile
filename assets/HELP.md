# User Help Guide

Welcome to the Authorized King James Version 1611 Bible App.

## Navigation
- **Books**: Swipe to the first tab or tap 'BOOK' in the top bar to select a book of the Bible.
- **Chapters**: Select a book first, then the second tab ('CHAPTER') will display available chapters.
- **Verses**: Select a chapter to see the verse grid. Tapping a verse number will jump to that verse in the Reader.

## Reading Views
You can switch between three distinct view styles in the **Settings** menu:
1. **Standard**: The pure text of the AKJV 1611 PCE.
2. **Superscript KJV**: Each word is numbered, matching the precise index used in the mathematical logic.
3. **Mathematics KJV 1**: A specialized view where connective words like "of" are replaced by mathematical grouping notation `()`.

## Audio Features
- **Play/Pause**: Use the controls at the bottom of the screen.
- **Sync**: The text highlights in real-time as the audio plays.
- **Jump to Word**: Tap any word in the reader to jump the audio precisely to that position.

## Selection & Copying
- **Word Range**: Long-press a word to start a selection, then tap another word in the same verse to select a range. Tap the Copy icon in the bar to copy to clipboard with its location reference.
- **Verse Range**: Tap the location reference (e.g., Gen1:1) on the left to select an entire verse. Tap multiple references to select a contiguous range.

## Mathematical Search
Tap the Search icon to access advanced functions:
- **Phrase Function**: Enter a location (e.g., `Gen1:1:1-5`) to see the specific phrase it represents.
- **Inverse Relation**: Enter a phrase to find every location where that exact sequence of words appears.

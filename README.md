### The 4-Vector Framework
Every word in the Bible is treated as a function of its Generalized Coordinates:
Word : BookChapter:Verse:Read ↦ Word(BookChapter:Verse:Read)
- **Book Index**: Height (H)
- **Chapter Index**: Depth (D)
- **Verse Index**: Length (L)
- **Read Index**: Breadth (B); word index within the Verse
- Word(BookChapter:Verse:Read) ≡ Word is a function "of" its Location = domain BookChapter:Verse:Read. 
- The parenthesis introduces "of"; and we have a 4-vector coordinate frame (H, D, L, B).

### Persistent Search & Inverse Relation Analysis
The App features an advanced Search Engine integrated into the Home Page for rapid vector mapping:
- **Persistent Context**: Search queries and result sets are cached automatically. You can exit the app and return to find your research context exactly as you left it until explicitly cleared.
- **Inverse Relation Mapping**: Automatically formats results into a mathematical relation: `Phrase ↦ { Loc1; Loc2; ... } RecordCount: n`.
- **Dynamic Output Filtering**: A real-time filter limits the Inverse Relation results, allowing for surgical isolation of specific occurrences while updating the `RecordCount` dynamically.
- **Surgical Copying**:
    - **Global Mapping**: A one-click copy icon captures the full Inverse Relation string.
    - **Targeted Clipping**: Each result entry features a copy icon that captures only the specific highlighted phrase to the clipboard for precise documentation.

### Study Hub & Analytical Tools
- **The Limit Table (GRID)**: Observes contextual convergence using `WTOTAG` (Where Two Or Three Are Gathered) and `CSTWS` (Comparing Spiritual Things With Spiritual) logic.
- **Surgical Dictionary Filtering**: Dictionary selections (DICT) dynamically filter the Limit Table, allowing the researcher to isolate specific spiritual relationships.
- **Occurrence-Based Accuracy**: The search engine tracks individual 4-Vector breadpoints (e.g., finding all 28+1 distinct occurrences of "charity" including the Preface).

### Advanced Selection & Export
- **Coordinate Linking**: Seamless verse selections are linked via underscores `_` while non-contiguous sets of words are separated with commas `,` for precise clipboard tracking.
- **Radiant "of" Operator**: The domain "of" a phrase is visually locked to preceding words with zero-space attachment to constantly remind us that these words are functions and conditional coloration to distinguish them from inherent text structure.

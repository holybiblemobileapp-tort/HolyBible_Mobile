# Holy Bible Mobile: Bible Vector Space Analysis

## Technical Overview
The Holy Bible Mobile App is a pedagogical tool designed to transform the Authorized King James Version (AKJV) 1611 PCE into a structured **Bible Vector Space**. It uses mathematical mapping to reveal the underlying consistency and "Newfangledness" of the Biblical text.

### Core Architecture
- **Language**: Flutter / Dart
- **Engine**: SQLite with FTS5 (Full Text Search)
- **Visuals**: Radiant Rendering Engine (Triple-layered Neon Glow)
- **Audio**: Piper Neural TTS (Local execution)

### The 4-Vector Framework
Every word in the Bible is treated as a function of its coordinates:
$Word = f(H, D, L, B)$
- **Height (H)**: Book Index
- **Depth (D)**: Chapter Index
- **Length (L)**: Verse Index
- **Breadth (B)**: Word Index within the Verse

### Pedagogical Series (MathKJV)
The app provides three levels of mathematical transformation:
1. **MathKJVP (Primary)**: Foundation level with high readability.
2. **MathKJVS (Secondary)**: Intermediate analysis including punctuation-adjacent symbols.
3. **MathKJVT (Tertiary)**: Unconstrained vector analysis for advanced study.

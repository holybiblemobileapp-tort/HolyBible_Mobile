# Holy Bible Mobile: Bible Vector Space Analysis

## Technical Overview
The Holy Bible Mobile App is a pedagogical tool designed to transform the Authorized King James Version (AKJV) 1611 PCE into a structured **Bible Vector Space**. It uses mathematical mapping to reveal the underlying consistency and "Newfangledness" of the Biblical text.

### Core Architecture
- **Language**: Flutter / Dart
- **Engine**: SQLite with FTS5 (Full Text Search)
- **Visuals**: Radiant Rendering Engine (Triple-layered Neon Glow)
- **Audio**: Piper Neural TTS (Local execution)

### The 4-Vector Framework
Every word in the Bible is treated as a function of its coordinates:
$Word = f(H, D, L, B)$
- **Height (H)**: Book Index
- **Depth (D)**: Chapter Index
- **Length (L)**: Verse Index
- **Breadth (B)**: Word Index within the Verse

### Pedagogical Series (MathKJV)
The app provides three levels of mathematical transformation:
1. **MathKJVP (Primary)**: Foundation level with high readability.
2. **MathKJVS (Secondary)**: Intermediate analysis including punctuation-adjacent symbols.
3. **MathKJVT (Tertiary)**: Unconstrained vector analysis for advanced study.

### Study Hub & Analytical Tools
The App features a dedicated **Study Hub** designed for deep vector analysis:
- **The Limit Table (GRID)**: Observes contextual convergence using `WTOTAG` (Where Two or Three Are Gathered) and `CSTWS` (Comparing Spiritual Things With Spiritual) logic.
- **Surgical Dictionary Filtering**: Dictionary selections (DICT) dynamically filter the Limit Table, allowing the researcher to isolate specific spiritual relationships.
- **Occurrence-Based Accuracy**: The search engine tracks individual 4-Vector breadpoints (e.g., finding all 28 distinct occurrences of "charity").
- **Curriculum Integration**: Direct access to the *"Your Strong Reasons"* pedagogical material via an integrated Markdown viewer.

### Advanced Selection & Export
- **Coordinate Linking**: Contiguous verse selections are linked via underscores `_` while non-contiguous sets are grouped with commas `,` for precise clipboard tracking.
- **Radiant "of" Operator**: Functional operators are visually locked to preceding words with zero-space attachment and conditional coloration to distinguish from inherent text structure.
